## HC
Freestanding C with LLVM.

### Targets:
| Binary Format | Architectures             | OS Support    |
| ------------- | ------------------------- | ------------- |
| Elf           | x86_64, aarch64, riscv64  | Linux         |
| Pe            | x86_64, aarch64, riscv64* | Windows, Uefi |
| Wasm          | wasm32                    | Wasi          |

\* Waiting for LLVM support.
